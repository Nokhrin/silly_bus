# vtable   
  
## Аксиома  
vtable | method table - структура данных в метаданных класса, которая содержит адресную таблицу переопределенных и непереопределенных методов  
Вызов метода посредством vtable осуществляет JVM для виртуальных методов  
  
## Прагматика  
Каждый класс имеет vtable  
При загрузке класса для наследованного класса создается копия vtable родителя, в которой переопределенные методы заменяются реализациями наследника  
Если метод переопределяет унаследованный метод, существует более одной реализации метода  
JVM читает фактический тип объекта и вызывает метод, который указан в vtable для этого типа   
=> экономия времени на последовательном поиске реализации в иерархии классов: O(1) вместо O(n)
