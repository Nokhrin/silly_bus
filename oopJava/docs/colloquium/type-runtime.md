runtime
время выполнения

runtime наступает после компиляции
runtime это совокупность компонентов, состав которых определяет яп и его реализация

runtime java включает

Где в runtime хранится информация о типе переменной
тип переменной - статический тип
является сущностью этапа компиляции,
в runtime не существует

тип объекта - динамический тип
JVM хранит объекты в куче
для объекта среди прочего хранится заголовок и указатель на class объект - klass object - он содержит информацию о родителях и интерфейсах, 
по этой информации проверяется тип в runtime
