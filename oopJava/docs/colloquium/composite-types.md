# Составные типы

> Составные типы (объекты, массивы, структуры)

конструктор типов - оператор в грамматике системы типов, аргументами которого являются атомарные и составные типы, 
и результатом работы которого является составной тип

произведение 
несколько значений одновременно
Class Person {int age, String name};

сумма - альтернатива
sealed interface Moving permits Walk, Run;

функция - отображение входа на выход
Function<int, String>

параметризация - тип, зависящий от параметра
List<char>


наследование не является конструктором типа

составной тип - тип, построенный с помощью конструктора типов из одного или нескольких других типов и который может быть синтаксически раскрыт на подтипы согласно грамматике системы типов


если тип имеет поля - это не атомарный, а составной тип
наличие только методов не делает тип составным


реализация в java
пустой класс - формально, логически атомарный
технически - в java имеет состояние - заголовок
class Empty {};

объекты с полями - результат произведения

массивы

обобщенные класы - результат параметризации

рассмотреть: aggregate types, structured types, complex types
