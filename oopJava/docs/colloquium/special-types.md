# Специальные типы данных

> Специальные типы данных - без данных - void в java, undefined в js, Unit/() в Scala

специальный тип - это единичный - имеющий одно значение - тип данных, используемый для обозначения отсутствия содержания

назначение
отличить процедуру (выполняет работу, не возвращает значение) от функции (выполняет работу, возвращает значение)

обеспечить однородность сигнатуры - каждая функция возвращает что-то

обеспечить единообразие синтаксиса системы типов - все вычисления имеют тип, спец тип - терминал отсутвия результата

описание различия функций с эффектами и чистых функций


реализация
спец типы - единичные типы - имеют одно значение в яп
примеры
Haskell - ()
java - void
js - undefined

технически различаются, семантически едины - метод что-то делает, ничего не возвращает

технически
() - единственное значение типа Unit
void - синтаксическая метка, не тип, нельзя создать значение типа void
undefined - маркер отсутствия инициализации
