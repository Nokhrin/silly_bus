Объяснить как связанны конструкция java try with resorces и тип auto closeable


тип auto closeable
интерфейс, определяет поведенческий контракт закрытия ресурса - предписывает метод close , отображаемый в эффект завершения, возвращает void
предназначен для гарантированного освобождения ресурсов, управляемых операционной системой или внешними системами - файлового дескриптора, сокета, транзакции бд

try with resorces
try (R r = expr) { body }
где значение переменной r - объект, реализующий интерфейс autocloseble
сокращенная форма записи try except + проверка на null, обеспечение корректного порядка закрытия ресурсов, обработку вложенных исключений, вызывающая метод close объекта r в блоке finally
конструкция позволяет компилятору гарантировать выполнение close при полном выполнении блока, при исключении, при раннем выходе из body

итог
autocloseble контракт -объект этого типа имеет эффект освобождение ресурса
try with resources - механизм исполнения контракта, позволяющий проверить намерение исполнения в compile time, перекладывая ответственность с программиста на компилятор
