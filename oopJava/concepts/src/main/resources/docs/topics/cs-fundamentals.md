### указатель на функцию

https://chat.qwen.ai/s/f3e0182f-41b5-4a33-ab72-c83991b308b1?fev=0.2.10

код для компьютера - данные
инструкции - последовательность байтов в ram

указатель на функцию - адрес ячейки памяти (например, 0x00400000), которая хранит первый символ кода функции

выражение `*somePointer` = оператор разыменовывания `*` + идентификатор указателя `somePointer`:
- возвращает содержимое памяти, находящееся по адресу, указанному в `somePointer`

`somePointer` - адрес 
`*somePointer` - данные по адресу

выражение `&someObject` = оператор взятия адреса `&` + идентификатор объекта `someObject`:
- возвращает адрес памяти, с которого начинается код объекта `someObject`
