## память процесса

для процесса определяется диапазон адресов в ram, называемый `память процесса`
память процесса разделена на функциональные блоки, в которых определенным образом хранятся 
данные статические (переданные в процесс глобальными переменными или параметрами исполняемой программы) - `.bss`, `.data`
данные динамические (создаваемые в runtime) - куча, стек
адреса функций (кода для выполнения процессором) - стек
скомпилированнй код, на который указывают адреса - `.text`

```
int main ()
{
  unsigned int x = 0x76543210;
  char *c = (char*) &x;  // зачем? почему недостаточно unsigned int x = 0x76543210; ? 
 
  if (*c == 0x10)
  {
    printf ("Underlying architecture is little endian. \n");
  }
  else
  {
     printf ("Underlying architecture is big endian. \n");
  }
 
  return 0;
}
```


объяснить отличие
инициализированные данные / `.data`
неинициализированные данные / `.bss`
