# Лямбда-выражения  

## Аксиоматическое определение | Идея | Что это?  
функция, которая имеет сигнатуру - типы значений параметров, имена параметров, тип возвращаемого значения, но не имеет имени  

## Прагматическое определение | Воплощение идеи | Для чего это?  
объект, выполняющий одну конкретную функцию с конкретной сигнатурой  
объект, который может быть передан как значение переменной  
требуется для передачи поведения без определения класса для этого поведения
