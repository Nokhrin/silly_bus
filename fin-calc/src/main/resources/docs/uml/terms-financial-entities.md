# Терминология предметной области

## 💳 Основные сущности

| Русское наименование |       Английский термин        |                                        Описание                                         |
|----------------------|--------------------------------|-----------------------------------------------------------------------------------------|
| Счет                 | `FinCalc.Account`              | Учетная единица, связанная с клиентом. Может быть расчетным, сберегательным, кредитным. |
| Владелец счета       | `FinCalc.Account Holder`       | Физическое или � юридическое лицо, на имя которого открыт счет.                         |
| Номер счета          | `FinCalc.Account Number`       | Уникальный идентификатор счета (обычно цифровой или алфавитно-цифровой).                |
| Счет клиента         | `Client FinCalc.Account`       | Счет, принадлежащий клиенту банка.                                                      |
| Состояние счета      | `FinCalc.Account Status`       | Состояние счета: `Active`, `Closed`, `Frozen`, `Pending`, `Suspended`.                  |
| Дата открытия счета  | `FinCalc.Account Opening Date` | Дата, когда счет был открыт.                                                            |
| Дата закрытия счета  | `FinCalc.Account Closing Date` | Дата, когда счет был закрыт.                                                            |

## 💰 Операции и транзакции

|   Русское наименование    |                 Английский термин                 |                       Описание                        |
|---------------------------|---------------------------------------------------|-------------------------------------------------------|
| Пополнение счета          | `Funding the FinCalc.Account` / `FinCalc.Deposit` | Внесение средств на счет.                             |
| Внесение средств          | `FinCalc.Deposit`                                 | Действие: `to deposit`.                               |
| Списание средств          | `FinCalc.Withdrawal`                              | Действие: `to withdraw`.                              |
| Перевод средств           | `FinCalc.Transfer`                                | Перемещение средств между счетом и счетом.            |
| Внутренний перевод        | `Internal FinCalc.Transfer`                       | Перевод внутри одного банка.                          |
| Внешний перевод           | `External FinCalc.Transfer`                       | Перевод в другой банк.                                |
| Банковский перевод (wire) | `Wire FinCalc.Transfer`                           | Быстрый международный перевод.                        |
| Транзакция                | `FinCalc.Transaction`                             | Любое финансовое действие, зафиксированное в системе. |
| История транзакций        | `FinCalc.Transaction History`                     | Список всех операций по счету.                        |
| Сумма транзакции          | `FinCalc.Transaction Amount`                      | Сумма операции.                                       |
| Дата транзакции           | `FinCalc.Transaction Date`                        | Дата, когда операция была совершена.                  |
| Время транзакции          | `FinCalc.Transaction Time`                        | Время совершения операции.                            |
| Статус транзакции         | `FinCalc.Transaction Status`                      | `Pending`, `Completed`, `Failed`, `Reversed`.         |
| Подтвержденная транзакция | `Confirmed FinCalc.Transaction`                   | Операция завершена и зафиксирована.                   |
| Ожидает обработки         | `Pending FinCalc.Transaction`                     | Операция в процессе обработки.                        |
| Отмененная транзакция     | `Cancelled FinCalc.Transaction`                   | Операция отменена до завершения.                      |

## 📊 Кредиты и займы

| Русское наименование |               Английский термин                |                        Описание                        |
|----------------------|------------------------------------------------|--------------------------------------------------------|
| Кредит               | `FinCalc.Loan`                                 | Заемные средства, возвращаемые с процентами.           |
| Кредитная карта      | `Credit Card`                                  | Средство для расчетов с кредитным лимитом.             |
| Кредитный лимит      | `Credit Limit`                                 | Максимальная сумма, которую можно занять.              |
| Остаток по кредиту   | `Outstanding Balance`                          | Сумма, которую нужно погасить.                         |
| Платеж по кредиту    | `FinCalc.Loan Payment` / `Installment Payment` | Платеж по кредиту по графику.                          |
| Ежемесячный платеж   | `Monthly Payment`                              | Платеж раз в месяц.                                    |
| Процентная ставка    | `Interest Rate`                                | Ставка, начисляемая на долг.                           |
| Проценты по кредиту  | `Interest Charges`                             | Сумма начисленных процентов.                           |
| Срок кредита         | `FinCalc.Loan Term`                            | Период, на который выдан кредит (в месяцах или годах). |
| Дата погашения       | `Maturity Date`                                | Дата окончания срока действия кредита.                 |
| Запрос на кредит     | `Credit Application`                           | Заявка на получение кредита.                           |
| Одобренный кредит    | `Approved FinCalc.Loan`                        | Кредит одобрен, но еще не выдан.                       |
| Отказ в кредите      | `FinCalc.Loan Declined`                        | Кредит не одобрен.                                     |
| Кредитный рейтинг    | `Credit Score`                                 | Показатель кредитоспособности клиента.                 |

## 📅 Даты и временные метки

|        Русское наименование         |     Английский термин      |                      Описание                      |
|-------------------------------------|----------------------------|----------------------------------------------------|
| Дата операции                       | `FinCalc.Transaction Date` | Дата, когда произошла операция.                    |
| Дата начисления                     | `Posting Date`             | Дата, когда операция была зафиксирована в системе. |
| Дата платежа                        | `Payment Date`             | Дата, когда деньги должны быть уплачены.           |
| Дата поступления средств            | `Funds Availability Date`  | Дата, когда средства становятся доступны.          |
| Дата окончания срока действия карты | `Card Expiry Date`         | Формат: `MM/YY` (например, `12/27`).               |
| Дата начисления процентов           | `Interest Posting Date`    | Дата, когда начислены проценты.                    |

## 📌 Дополнительные термины

|     Русское наименование     |         Английский термин         |                             Описание                             |
|------------------------------|-----------------------------------|------------------------------------------------------------------|
| Баланс счета                 | `FinCalc.Account Balance`         | Текущая сумма на счете.                                          |
| Дебетовый счет               | `Debit FinCalc.Account`           | Счет, с которого списываются средства.                           |
| Кредитный счет               | `Credit FinCalc.Account`          | Счет, на который поступают средства (например, кредитная карта). |
| Сумма списания               | `FinCalc.Withdrawal Amount`       | Сумма, снятая со счета.                                          |
| Сумма пополнения             | `FinCalc.Deposit Amount`          | Сумма, внесенная на счет.                                        |
| Комиссия за перевод          | `FinCalc.Transfer Fee`            | Плата за выполнение перевода.                                    |
| Комиссия за снятие наличных  | `Cash FinCalc.Withdrawal Fee`     | Плата за снятие наличных со счета.                               |
| Система мгновенных переводов | `Instant FinCalc.Transfer System` | Например, SEPA Instant, Zelle, Faster Payments.                  |

```markdown
### FinCalc.Transaction Type

- `DEPOSIT`
- `WITHDRAWAL`
- `TRANSFER`
- `INTEREST_CREDIT`

- `REFUND`

- `ACTIVE`
- `CLOSED`
- `FROZEN`
- `PENDING`
- `SUSPENDED`

- `PENDING`
- `COMPLETED`
- `FAILED`
- `REVERSED`
- `PENDING_REVERSAL`
```
