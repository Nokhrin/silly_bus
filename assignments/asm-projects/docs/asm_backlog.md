# asm-string/docs/specification.md

## Учебный проект `asm-string` - 16 часов

### Цель
Понимание работы со строками, реализация стандартных функций C, уязвимости переполнения буфера, защиты памяти (DEP/NX, ASLR, Stack Canary).

### Задача
Реализовать строковые функции с проверкой границ, продемонстрировать уязвимости и защиты.

### Тех. ограничения
- Linux x86-64, NASM синтаксис
- Без libc (только syscall write/exit_group)
- System V AMD64 ABI
- Сборка с `-z execstack` только для демонстрации уязвимостей

---

## Почему RISC-V для os-bootloader?

| Фактор       | x86                              | RISC-V                    |
|--------------|----------------------------------|---------------------------|
| Спецификация | Проприетарная, сложная           | Открытая, модульная       |
| Режимы       | Real, Protected, Long            | Machine, Supervisor, User |
| Прерывания   | APIC, PIC, сложная маршрутизация | PLIC, простая модель      |
| Документация | Разрозненная, NDA                | Единая, доступна          |
| Эмуляция     | QEMU, Bochs (сложно)             | QEMU (проще)              |
| Обучение     | Legacy, историческое             | Современное, чистое       |
