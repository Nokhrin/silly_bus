# Учебный проект `os-bootloader` — 40+ часов (факультатив)

## Цель

Понимание загрузки ОС: Real Mode, Protected Mode, GDT, paging, Ring 0/3.

## Задача

Реализовать загрузчик с переходом в protected mode и базовой обработкой прерываний.

## Тех. ограничения

- x86 (32-bit), NASM синтаксис
- Без ОС (bare metal)
- Эмуляция в QEMU/Bochs

---

## Этап 1: Real Mode (10 часов)

| Параметр | Значение |
|----------|----------|
| Задача | Загрузочный сектор, 512 байт, сигнатура 0xAA55 |
| Концепции | BIOS прерывания (int 0x10, int 0x13), сегментация |
| Проверка | Вывод символа на экран, чтение сектора |

---

## Этап 2: Protected Mode (10 часов)

| Параметр | Значение |
|----------|----------|
| Задача | Переход в protected mode, GDT |
| Концепции | Сегменты кода и данных, селекторы |
| Проверка | Выполнение кода в 32-bit режиме |

---

## Этап 3: Paging (10 часов)

| Параметр | Значение |
|----------|----------|
| Задача | Таблицы страниц, CR3, виртуальная адресация |
| Концепции | PDE, PTE, включение paging (CR0.PG = 1) |
| Проверка | Identity mapping, доступ к памяти по виртуальным адресам |

---

## Этап 4: Ring 0/3 (10 часов)

| Параметр | Значение |
|----------|----------|
| Задача | TSS, call gates, пользовательский режим |
| Концепции | Переключение стека при переходе в Ring 0, системные вызовы |
| Проверка | Запуск кода в Ring 3, изоляция пользовательских программ |
