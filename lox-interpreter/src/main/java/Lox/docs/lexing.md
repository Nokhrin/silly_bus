# Лексический анализ

## Лексемы и токены

- считывание последовательности символов
- определение минимальных - атомарных - подпоследовательностей, имеющих смысл по правилам анализируемого языка
- лексема - такая подпоследовательность

для выполнения лексического анализа удобно определить множество связок/мэппингов имен допустимых лексем к их определениям 
при анализе определять, входит ли считанная лексема в множество определенных
принимать решение о следующем действии
другими словами - 
- проверяем из определенного множества 
  - значит, известно максимальное количество проверок, не требуется проверять все возможные последовательности и "угадывать" значение
  - это повышает производительность, или, по крайней мере, гарантирует, что программа завершится
- меньше работы и сложнее ошибиться

## Регулярные языки и регулярные выражения
правила группировки последовательности символов в лексемы - правила лексической грамматики
формальные грамматики определяют языки, которые можно отнести к регулярным

> в реализации Lox не используются регулярные выражения, реализуется посимвольный сканер
