### Блок 6.2: Spinlock реализация (4 часа)

```nasm
spinlock_lock:
    mov eax, 1
    .wait:
    lock xchg [rdi], eax  ; атомарный обмен
    test eax, eax
    jnz .wait             ; если было занято - ждать
    ret

spinlock_unlock:
    mov dword [rdi], 0    ; освободить (не атомарно, но безопасно)
    ret
```

### Сценарий

1. BSP (Bootstrap Processor) инициализирует LAPIC
2. BSP отправляет SIPI (Startup IPI) всем AP
3. AP запускают код по адресу 0x1000
4. Каждое ядро инициализирует свой стек (per-CPU data)
5. Счётчик активных ядер атомарно инкрементится
