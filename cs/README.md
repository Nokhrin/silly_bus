## Символы логических операторов (по Gries)

| Символ | Название                | Приоритет | Аналог в коде       |
|--------|-------------------------|-----------|---------------------|
| `¬`    | отрицание (not)         | высший    | `!` / `not`         |
| `∧`    | конъюнкция (and)        | высокий   | `&&` / `and`        |
| `∨`    | дизъюнкция (or)         | средний   | `\|\|` / `or`       |
| `⇒`    | импликация (implies)    | низкий    | `a ⇒ b` ≡ `¬a ∨ b`  |
| `⇐`    | обратная импликация     | низкий    | `a ⇐ b` ≡ `b ⇒ a`   |
| `≡`    | эквивалентность (iff)   | низший    | `==` для bool       |
| `≠`    | неэквивалентность (xor) | низший    | `!=` для bool       |
| `↦`    | замена                  | -         | отображение/маппинг |
