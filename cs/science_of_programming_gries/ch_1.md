## 3. Переведите следующие предложения на язык логики высказываний:

(a) x < y или x = y.  
p ≡ (x<y)
q ≡ (x=y)
p ∨ q

(b) либо x < y, x = y, либо x > y.  
p ≡ (x<y)
q ≡ (x=y)
r ≡ (x>y)
p ∨ q ∨ r

(c) Если x > y, а y > z, то u = w.  
p ≡ (x>y)
q ≡ (y>z)
r ≡ (u=w)
(p ∧ q) ⇒ r

(d) Все следующие утверждения истинны: x < y, y < z и u = w.  
p ≡ (x<y)
q ≡ (y<z)
r ≡ (u=w)
p ∧ q ∧ r

(e) Самое большее одно из следующих утверждений истинно: x < y, y < z, u = w.  
p ≡ (x<y)
q ≡ (y<z)
r ≡ (u=w)
¬(p ∧ q) ∧ ¬(p ∧ r) ∧ ¬(r ∧ q)

(f) Ни одно из следующих утверждений не истинно: x < y, y < z, u = w.  
p ≡ (x<y)
q ≡ (y<z)
r ≡ (u=w)
¬p ∧ ¬q ∧ ¬r == ¬(p ∧ q ∧ r)

(g) Следующие утверждения не являются истинными одновременно: x < y, y < z, u = w.  
p ≡ (x<y)
q ≡ (y<z)
r ≡ (u=w)
¬p ∨ ¬q ∨ ¬r

(h) Когда x < y, тогда y < z; когда x ≥ y, тогда u = w.
p ≡ (x<y)
q ≡ (y<z)
r ≡ (u=w)
(p ⇒ q) ∧ (¬p ⇒ r)
((not p) or q) and (not(not p) or r)

(i) Когда x < y, y < z означает, что u = w, 
но если x ≥ y, то y < z не может выполняться; 
однако если u = w, то x < y.  
p ≡ (x<y)
q ≡ (y<z)
r ≡ (u=w)
((p ⇒ (q ⇒ r)) ∧ (¬p ⇒ ¬q)) ∧ (r ⇒ p)

(j) Если выполнение программы P начинается при x < y, 
то оно завершается при y = 2x.  
p ≡ (x<y)
q ≡ (y=2x)
s ≡ начало выполнения программы P
t ≡ завершение выполнения программы P
(p ∧ s) ⇒ (q ∧ t)

(k) Выполнение программы P, начавшееся при x < 0, не завершается.
p ≡ (x<0)
s ≡ начало выполнения программы P
t ≡ завершение выполнения программы P
(p ∧ s) ⇒ ¬t

## 4. Ниже приведены несколько предложений. Обозначьте идентификаторами элементарные части (например, «живут как кошка с собакой») и переведите предложения на язык высказываний.

(a) Будет или не будет дождь, я пойду купаться.  
p ≡ "идет дождь"
q ≡ "иду купаться"
((p ∨ ¬p) ⇒ q) ≡ q

(b) Если будет дождь, я не пойду купаться.  
p ≡ "идет дождь"
q ≡ "иду купаться"
p ⇒ ¬q

(c) Живут как кошка с собакой.  
p ≡ "Живут как кошка с собакой"
p

(d) Живут, как кошка или собака.  
p ≡ "Живут как кошка"
q ≡ "Живут как собака"
p ∨ q

(e) Если дождь будет лить как из ведра, то будь я проклят, купаться не пойду.  
p ≡ "дождь льет как из ведра"
q ≡ "иду купаться"
p ⇒ ¬q

(f) Если, когда я буду купаться, дождь будет лить как из ведра, то будь я проклят.
p ≡ "дождь льет как из ведра"
q ≡ "выполняется купание"
r ≡ "я проклят"
(p ∧ q) ⇒ r
